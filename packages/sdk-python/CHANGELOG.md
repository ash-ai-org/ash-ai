# ash-ai-sdk (Python)

## 0.0.6 - 2026-02-24

### Added

- Auto-generated Python SDK from OpenAPI spec using openapi-python-client
- Synchronous and async client support via httpx
- Full API coverage: agents, sessions, credentials, queue, attachments, usage, health
- Typed request/response models with attrs
- PyPI publishing via trusted publisher (OIDC)

### Changed

- Package renamed from `ash-ai` to `ash-ai-sdk` (namespace conflict on PyPI)
